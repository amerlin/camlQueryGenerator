# Caml Query Generator
Javascript Camel Query Generator
This function is really pretty simple

Caml Query format 

```
<Where>
  <Eq> Condition... </Eq>
  <And>
     <Eq> Condition ... </Eq>
     <And>
        <Eq> Condition ... </Eq>
        <Eq> Condition ... </Eq>
     </And>
  </And>
</Where>
```
